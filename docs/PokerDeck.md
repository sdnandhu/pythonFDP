# PokerDeck


Represents a deck of cards that can deal poker hands. 

## Methods


### deal_hands


Deals hands from the deck and returns Hands.   
num_cards: cards per hand num_hands: number of hands   
returns: list of Hands 

#### Parameters
name | description | default
--- | --- | ---
self |  | 
num_cards |  | 5
num_hands |  | 10
